# 张景皓的成长时光机 (Baby Full Month Site)

这是一个为庆祝宝宝满月而设计的网站，采用 Next.js 14 开发。

## 特色功能
- **流星雨 & 樱花特效**：全屏动态背景，营造梦境感。
- **照片墙**：自动从 GitHub 仓库同步照片，支持瀑布流展示。
- **投屏模式**：全屏幻灯片自动轮播，支持播放/暂停及全屏切换。
- **响应式设计**：适配手机、平板及电脑显示。

## 技术栈
- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Framer Motion** (动画特效)
- **Ma Shan Zheng** (书法字体)

## 部署指南 (Vercel)
1. 在 Vercel 中导入此项目。
2. 配置环境变量 `GITHUB_TOKEN`，用于访问私有照片仓库。
3. 部署成功后即可访问。
